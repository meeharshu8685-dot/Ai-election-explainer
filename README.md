# Election Copilot 🗳️

Election Copilot is an intelligent, neutral, and highly interactive assistant designed to help users navigate the complexities of elections. Built for the **Google Antigravity Challenge**, it leverages **Gemini 1.5 Pro** to provide personalized guidance, simulations, and fact-checking.

## 🚀 Core Features

- **🎮 Interactive Election Simulator**: Step into the shoes of a voter, candidate, or officer.
- **🗺️ Personalized Election Timeline**: Real-time tracking of election milestones in India.
- **🧩 Step-by-Step Guided Mode**: Simplified wizards for registration and voting day.
- **⚖️ Misinformation Detector**: AI-powered fact-checking for election-related claims.
- **💬 Context-Aware Q&A**: Intelligent responses tailored to your persona (Beginner, Student, Expert).

## 🛠️ Tech Stack

- **Frontend**: Next.js 15 (App Router), Tailwind CSS, Framer Motion.
- **AI Layer**: Google Gemini 1.5 Pro via Vercel AI SDK.
- **Styling**: Premium Glassmorphism with a Vibrant Blue & Emerald theme.

## 🧠 Master System Prompt

The assistant is governed by a sophisticated system prompt that ensures neutrality, clarity, and engagement. It adapts its tone based on the user's selected persona and complexity level.

## 🏃 Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set up your `.env.local` with `GEMINI_API_KEY`.
4. Run the development server: `npm run dev`.
